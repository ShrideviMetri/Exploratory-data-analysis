# Exploratory-data-analysis
Objective
The objective of this project is to perform Exploratory Data Analysis on employee data and identify patterns, trends, and relationships among variables.
Dataset Description
The dataset contains employee information including:

- Employee ID
- Age
- Salary
- Experience
- Department

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

Analysis Performed

1. Data loading and inspection
2. Statistical summary
3. Missing value analysis
4. Duplicate record analysis
5. Correlation analysis
6. Age distribution analysis
7. Salary distribution analysis
8. Experience distribution analysis
9. Age vs Salary analysis
10. Experience vs Salary analysis
11. Department-wise employee analysis

Findings

- No missing values were found in the dataset.
- No duplicate records were detected.
- Salary increases as experience increases.
- Employees with higher age generally receive higher salaries.
- Finance department contains some of the highest-paid employees.
- Positive correlation exists between Age and Salary.
- Positive correlation exists between Experience and Salary.
- The dataset is well-structured and suitable for analysis.

Conclusion

The analysis shows that employee salary is strongly influenced by experience and age. Employees with more years of experience generally receive higher salaries. The dataset provides useful insights into workforce characteristics and compensation trends.

Outcome

This project demonstrates the use of Exploratory Data Analysis techniques to understand data, identify trends, and generate meaningful business insights.
